Automation Lab: Robots Learning from Humans Undergraduate Opening

Information provided by Michael Laskey, 10/1/15


Project Description:

Recent algorithmic advances in learning from demonstration has created a way to learn different tasks without any prior information. These model free approaches have been shown to be successful in a number of applications such as playing video games, flying a quadcopter and recently learning surgical subtasks. We are interested in studying both adaptations to the machine learning algorithms and the psychology of teaching. This work builds on our most recent paper submission: http://berkeleyautomation.github.io/shiv/

Students working on the project will help with:

  1. Performing Human Robot Interaction experiments
  2. Designing Experiments to help test different learning algorithms
  3. Integrating deep learning architectures into current experiments
  4. Launching experiments in the cloud to learn on a large dataset
  5. Creating environments for learning in physics simulator
  6. Analyzing the theoretical implications of the learning algorithms

This work will help us answer the future research questions:

  • How should a human provide feedback to a robot?
  • Can a student robot teach the human new skills?
  • Can a robot learn in simulation first and then work with a human in the real world?

We have a successful history of mentoring undergraduates so far. Our first set of undergraduate researchers got working on an independent project within a year, and their work later appeared in the New York Times. Dex-Net 1.0 was also largely built by undergraduate researchers. We encourage applicants of all backgrounds to apply!
